## 📁 项目结构

```
Spiral-Up/
├── scripts/           # 实用脚本工具
│   └── 切书神技.zsh   # 智能EPUB切书工具
├── docs/              # 项目文档
│   └── README.md      # 详细使用说明

```

## 🚀 快速开始

### 📚 EPUB切书工具

智能识别章节标题，自动命名文件的电子书切分工具：

```bash
chmod +x scripts/切书神技.zsh
./scripts/切书神技.zsh 'your-book.epub'
```

详细文档：[scripts/README.md](docs/README.md)
